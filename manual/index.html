Configuración del Entorno

1

Instalar Extensión del IDE

Habilita el puente de comunicación instalando nuestra extensión oficial en tu IDE preferido (Cursor, Antigravity o VSCode). Permite la gestión remota de forma segura y eficiente.

cd vscode-extension
# Instalar mediante .vsix o marketplace
2

Tema Visual Obligatorio

Para una experiencia óptima y profesional, configura el IDE con un Tema Oscuro de Alto Contraste. Esto garantiza la legibilidad y coherencia visual con la interfaz de control.

Configuración > Tema de color > Dark High Contrast
3

Arranque Standalone

Ejecuta el script unificado para iniciar tanto el servidor de control como la interfaz web en segundos.

.\start-standalone.ps1
